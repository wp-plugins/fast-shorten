== Installation ==

### Installation

1. Upload 'fast-shorten' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

### Shortcode

1. [fastshorten url="" text="" target="" rel=""]

### [fastshorten] Attributes

1. url="paste long url here to shorten it" (required attributes).
2. text="enter your text, example: Download" this is attributes for display your text (optional attributes).
3. target="enter your custom target, example: _self" this is attributes to change link target, defaut target is _blank (optional attributes).
4. rel="enter your custom rel, example: follow" this is attributes to change link rel, defaut rel is nofollow (optional attributes).

### Example

	This is my [fastshorten url="https://wordpress.org/plugins/responsive-full-background/" text="short link (hover me plz)"] with text attribute.
	This is my another short link [fastshorten url="https://wordpress.org/plugins/perso-slider/"] without text attribute.
